Music Store REST API

REST API dla sklepu muzycznego, umożliwiające przeglądanie dostępnych albumów, zarządzanie ofertą i stanami magazynowymi.

Opis Autoryzacja Endpointy Model danych Dokumentacja

Opis

Aplikacja REST API pozwala na zarządzanie bazą albumów muzycznych. Goście mogą przeglądać listę albumów, a pracownicy po zalogowaniu (JWT) mogą zarządzać treścią.
